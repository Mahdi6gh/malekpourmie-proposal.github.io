# SEO, GEO & UX Growth Proposal for malekpourmie.net

## Overview

This repository contains a professional SEO, GEO (Generative Engine Optimization), UX, and Digital Growth Proposal prepared for **malekpourmie.net**, one of the leading immigration consulting brands in the Persian-speaking market.

The purpose of this proposal is to identify growth opportunities, technical issues, user experience challenges, and strategic improvements required to increase organic visibility, conversion rates, brand authority, and long-term digital performance.

---

## Project Goals

- Improve Technical SEO infrastructure
- Enhance User Experience (UX) and Navigation
- Optimize Information Architecture
- Increase Conversion Rate Optimization (CRO)
- Strengthen Internal Linking Structure
- Improve Multilingual SEO (Persian, English, Arabic)
- Prepare content for AI Search Engines (GEO)
- Increase Brand Authority and Trust Signals
- Improve Accessibility for Iranian Users
- Develop a scalable Digital PR and Link Building strategy

---

## Key Findings

### User Experience Issues

- Overloaded header structure
- Complex navigation paths
- Poor accessibility to important landing pages
- Inconsistent page layouts
- Weak conversion funnel implementation

### Technical SEO Issues

- Broken internal links
- 404 errors
- Orphan pages
- Redirect loops
- Weak anchor text structure
- Improper heading hierarchy (H1-H6)

### Content & Information Architecture

- Need for stronger Pillar & Cluster strategy
- Weak internal linking between related topics
- Service pages built using blog-like structures
- Content optimization opportunities for GEO and AI search

### Multilingual Challenges

- Arabic version requires content restructuring
- English version requires quality improvements
- Need for proper multilingual SEO implementation

### Accessibility Challenges

One of the most critical issues identified is the limited accessibility of the website for users inside Iran due to networking and routing restrictions.

Implementing a professional GeoDNS solution and optimized regional routing can significantly improve accessibility, user retention, and conversion rates.
